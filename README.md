# Cara Menjalankan Aplikasi
1.	Download dan install JDK (https://adoptium.net/temurin/releases) pada komputer anda.
2.	Download dan install aplikasi Greenfoot (https://www.greenfoot.org/download) pada komputer anda.
3.	Download source code dalam bentuk zip kemudian extract.
4.	Buka aplikasi Greenfoot.
5.	Klik Scenario pada kiri atas.
6.	Pilih Open.
7.	Pilih folder source code yang tadi sudah di extract.
8.	Klik Run.
